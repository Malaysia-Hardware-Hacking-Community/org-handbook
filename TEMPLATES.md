# Member Introduction Template

Use the following format when introducing yourself in the Discord server after completing onboarding.

## Copy-Ready Template

```text
Nickname: (what you want people to call you, can be an alias)
Background: (e.g., Student, Engineer, Hobbyist, Pentester, Researcher, etc.)
Experience Level: (Beginner, Intermediate, or Advanced)
Field of Interest: (CTFs, Research, Modification, or any combination)
What Brought You Here: (what you're hoping to learn or work on)
LinkedIn: (profile link or name, optional)

P/S: (anything else you want to share, delete if none)
```

## Example 1

```text
Nickname: wr3nch
Background: Embedded Systems Engineer
Experience Level: Intermediate
Field of Interest: Research, Modification
What Brought You Here: Want to get into firmware security and start contributing PoCs
LinkedIn: https://www.linkedin.com/in/your-profile
```

## Notes

- Keep it concise.
- You can use an alias. No need to give your real name.
- Don't post sensitive personal details.
- Share only links and identifiers you're comfortable making visible to all server members.
